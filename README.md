## Description du TP3
Ce projet consiste à analyser une partie du Yelp Dataset afin de créer un dataset intégré regroupant des informations provenant de plusieurs tables (restaurants, avis, utilisateurs, etc.)

À partir de ce dataset, nous avons dû :
* Nettoyoyer et préparer les données
* Construire un dataset final (restaurants_features.csv) avec toutes les colonnes demandées
* Produire deux visualisations (1 statiques, 1 interactive)
* Effectuer une analyse exploratoire des interprétation des graphiques ainsi que des observations générales clées
* Documenter le tout dans un notebook
